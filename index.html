01. 台車點檢系統 (Web App)

Pushcart Inspection System

利用 HTML5 與 QR Code 技術，解決傳統紙本點檢的效率與保存問題。實現即時資料庫串接與視覺化管理。

  • 架構： GitHub Pages 託管前端 + Google Sheets 作為後端資料庫 (GAS)。
  • 特點： 免安裝 APP (Web-based)、0.1秒極速掃描、紅綠燈號視覺化提示。
  • 效益： 點檢資料結構化，維護單位可即時查詢維修部位，工安人員可快速複查稽核。
HTML5 Google Apps Script QR Code Scanner

02. 人機介面 YOLO 偵測系統

YOLO-Based HMI Detection

運用邊緣運算技術，自動辨識機台人機介面 (HMI) 的設定參數，防止人為設定錯誤導致的產品報廢。

  • 核心技術： 樹莓派 (Raspberry Pi) 運行 YOLO 模型，建立封閉式 Wi-Fi 網路確保資安。
  • 運作流程： 手機連線 -> 拍攝畫面 -> 樹莓派運算 -> 自動判斷條件 -> 自動截圖存證。
  • 突破： 解決手機算力不足問題，老舊設備也能使用；照片容量從 15MB 降至 100KB。
YOLOv8 Raspberry Pi Python Flask

03. 軟板皺褶檢測 2.0

FPC Wrinkle Detection

基於 OpenCV 的機器視覺系統，即時監控生產線上的軟板皺褶情況，提供數據化分析。

  • 演算法： Sobel 邊緣檢測 + 二值化處理 + 輪廓分析。
  • 功能： 即時繪製折線圖、異常自動截圖、LOG 檔斷電保護寫入。
  • 應用： 準確抓出造成皺褶的特定夾具編號，協助維護單位精準維修。
OpenCV Python Sobel Edge Data Visualization
